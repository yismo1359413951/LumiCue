# 游戏化隐形提词器 设计文档

> 2026-06-24 · 靓相 Shotlit · 让 Mac 录屏博主念稿像玩游戏，眼前一亮、不同质化。

## 目标
给 Mac 录屏+露脸博主一个**好玩、不普通**的隐形提词器：念稿过程像玩游戏（有节奏感/成就感/沉浸感），而录屏/直播的观众完全看不到它。

## 核心约束
- **隐形**：`NSWindow.sharingType = .none`（系统级从录屏/直播排除）。已验证可行。
- **不固定刘海**：提词器要念稿（大块文字），不能塞顶部刘海小条（空间小 + Vibe Coding 时菜单栏图标多）。做成**自由浮动岛**：默认摄像头附近，可拖到屏幕任意位置 + 任意调大小。
- **借鉴 Vibe Island 的质感与聚焦理念**（毛玻璃精致、只突出关键），但**不照搬固定刘海形态**。

## 视觉效果（按实现顺序）

### 第 1 批：地基（先做、先验）
1. **毛玻璃背景** `NSVisualEffectView` — 后面窗口内容被模糊化（不透出清晰内容叠字），文字浮其上清楚。这是治"之前透明叠字乱/纯黑挡视野/描边被否"的根本解。
2. **舞台追光（当前行）** — 当前念的行最亮，上下行渐暗。径向渐变/透明度遮罩实现。
3. **3D 纵深** — 当前行最大最近，上下行近大远小往里缩 + 虚化。每行 CALayer 的 transform（scale + 透明度按距当前行的距离）。
4. **平滑滚动** — 逐像素平滑（已有），配合行的纵深/追光过渡。

### 第 2 批：游戏化（地基验过再加）
5. **火箭进度赛道** — 底部一条跑道，小火箭随念稿进度前进，念完冲到终点 🏁。进度 = 当前滚动位置 / 总高度。
6. **卡拉 OK 逐字点亮** — 当前行的字随念逐个高亮扫过（按速度推进的逐字 attributes 动画）。
7. **念过即飘散** — 滚出顶部的行柔和淡出。
8. **小精灵陪读** — 一个动画小角色沿文字侧边"走"，陪念（最后做，可选）。

## 已有功能（保留）
自动滚动 + 速度 5 档 + 字号数值列表(10–56pt) + 导入 txt + 控制条(小/中/大/清/✕/暂停) + 字色选择。

## 新增：字体可选
列出 Mac 所有系统字体（`NSFontManager.shared.availableFontFamilies`）供选，右键菜单。后续可加 GitHub 开源字体（放进 bundle）。

## 技术栈
- 毛玻璃：`NSVisualEffectView`(.hudWindow/.underWindowBackground)
- 3D 纵深 + 追光：放弃单一 NSTextView 整体滚动，改**逐行渲染**（每行一个 CATextLayer/NSTextField），按"距当前行的距离"算 scale/透明度/模糊 → 实现纵深 + 追光。
- 火箭进度/飘散：`CABasicAnimation` / layer 动画。
- 卡拉 OK：当前行用 `NSAttributedString` 逐字改 foregroundColor。
- 隐形：`sharingType = .none`（已有）。

## 风险/诚实说明
- 从"单 NSTextView 滚动"改成"逐行 layer 渲染"是较大重构（3D纵深/追光/逐字都需要逐行控制）。分批做、每批编译+界面验证。
- 第 1 批（毛玻璃+追光+3D纵深）是地基，做好验过再上游戏化效果，避免一次堆太多翻车。

## 验收
每批做完：编译 BUILD SUCCEEDED + 启动 + 用户肉眼看效果确认（我不替判断好看）。
