## 🔑 决定性发现：基座的窗口排除机制天然适配我们

Snapzy 录屏时**默认排除自己 App 的所有窗口**（避免录进工具栏），再用 `exceptingWindows` 把需要露出的浮层（鼠标高亮/标注/键盘）"加回来"让它出现在录像里。证据：
- `MouseClickHighlightWindow.swift` 注释原文：*"Captured by ScreenCaptureKit via exceptingWindows so the effect appears in the recorded video."*
- `RecordingCoordinator.swift:1163` `await recorder.addExceptedWindow(windowID: highlightWindow.overlayWindowID)`
- 每个浮窗有 `overlayWindowID: CGWindowID { CGWindowID(windowNumber) }`

**对我们的意义（几乎免费）：**
- **露脸 bubble**：建浮窗 + 调 `recorder.addExceptedWindow(...)` → 出现在录像里 ✅
- **隐形提词器**：建浮窗，**不调** addExceptedWindow → 作为 Snapzy 窗口被默认排除 → **自动录不进去** ✅（命根子几乎零成本）

## 浮窗标准模式（仿 MouseClickHighlightWindow）
```
NSWindow 子类, styleMask=.borderless
isOpaque=false; backgroundColor=.clear; hasShadow=false
level = .floating + 1   // 置顶
collectionBehavior = [.canJoinAllSpaces, .fullScreenAuxiliary]
ignoresMouseEvents = true   // bubble/提词器改 false 以便拖动/交互
```
