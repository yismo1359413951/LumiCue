## 二、用户痛点（国内外，按痛感）
1. 🔥🔥🔥 **被看出在念稿 / 眼神飘 / 不真诚**（国内外第一痛点）。眼神现状：只有 BIGVU 做，还是**后期AI修**，没人**实时**治。
2. 🔥🔥🔥 **语音跟随都不稳**：PromptSmart 用户原话"要么跳到前面、要么直接停摆"、"voice control; that is a joke"；其它都依赖安静环境+好麦才准。**没人把语音跟随做到真稳**。
3. 🔥🔥🔥 **卡壳/反复重录的心理压力**："1分钟录30遍"——**完全空白，没人碰**。
4. 🔥🔥 **订阅乱收费**：国内实名投诉扣792/168元；国外录一条前先锁订阅。**没人做开源**。
5. 🔥🔥 **提词器被录进画面** → 隐形（但已被几家占）。
6. 🔥 整段铺屏→眼神乱跳；工作流断裂（录完切App编辑/字幕/发布）；工程质量（闪退/安卓阉割/广告）。

## 三、突破口（我们怎么在竞品之上更好）
> 隐形已经不稀奇了。真正能赢的是把下面**没人做好的**做到位：

- **A. 语音跟随做到"真稳"** 🔴最硬护城河
  直接打 PromptSmart 最大软肋（跳读/停摆）。本地识别 + 抗噪 + 抗口音 + **卡壳/即兴自动接住**（不跳读不停摆）。竞品全在这翻车 = 我们的机会。
- **B. 实时治"眼神飘"** 🔴头号痛点正面突破
  短句喂词（一次只贴一两句到镜头光轴）+ 视线锚点 → 眼睛几乎不动。BIGVU 只能后期修，**我们实时**。
- **C. 卡壳救场 / 反NG心理** 🔴空白市场
  治"录30遍"。卡住稳稳停 + 给下一句关键词 + 一条过的正反馈。没人做。
- **D. 开源免费** 🔴信任突破
  对比 ShareSpeak/Notchie 买断、VODIUM/BIGVU 订阅，**开源免费**国内外通吃，砍掉收费怒点。
- **E. 设计大气** 🟡
  竞品普遍工具感/朴素，高级克制的设计语言（参考 Linear/Raycast）本身就是差异。去掉幼稚元素（小猫/烟花）。

**一句话定位**：别人是"隐形提词器"，我们是 **"让你像没看稿一样自然开口的隐形提词器"** —— 语音跟随做到真稳 + 实时治眼神飘 + 卡壳兜底 + 开源。
