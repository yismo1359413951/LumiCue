# 靓相 Shotlit · 调研与技术方案

> 本文固化所有决策依据，防长对话遗忘。新对话接手先读这份 + tasks/todo.md。

## 一、产品定位
市场是割裂的两半：**会美颜的不会录屏**（CapCut/直播工具），**会录屏的没美颜**（Screen Studio/Loom）。
没人把"自动美化录屏 + 实时美颜露脸 + 异形框 + 隐形提词器"合在一起 —— 这是护城河。
时机：Loom 被 Atlassian 收购后口碑崩盘+涨价；Screen Studio 没 Windows、取消买断转订阅。

## 二、命名决策
- 中文：**靓相**（靓=美，相=相机/相貌/亮相，一看就懂"美美地出镜"，好注册商标）
- 英文：**Shotlit**（Shot=录屏截图，呼应 CleanShot 血统；lit=点亮/俚语"超赞"，呼应"靓"）

## 三、基座决策：Snapzy（不是 CleanShot 本体）
- ⚠️ 关键澄清：CleanShot X 是闭源商业软件。用户给的 Snapzy、better-shot 是它的**开源仿制品**。
- 选 Snapzy（https://github.com/duongductrong/Snapzy）理由：BSD-3 可商用二开、1.5k star、非常活跃（v1.24）、模块化彻底（17 个 Service）、已有截图/录屏/标注/视频编辑/云上传/OCR/多语言。better-shot 功能少、仍 0.x，落选。
- 技术栈：Swift + SwiftUI + AppKit + **ScreenCaptureKit** + Vision + Sparkle。macOS 13+。

## 四、美颜库调研结论（用户给的 6 个）
| 库 | 平台 | 有瘦脸大眼? | License | 可复用什么 |
|---|---|---|---|---|
| MagicCamera | 安卓 Java/GLSL | ❌ 仅磨皮美白滤镜 | 无 License(风险) | GLSL shader 参考 |
| BeautyCamera | Python 桌面 | ❌ | MIT | OpenCV 思路(CPU 慢,不适合实时) |
| AwemeLike | iOS OC/GLSL | ✅ 瘦脸大眼(靠 Face++ 收费) | MIT | 瘦脸/磨皮 GLSL shader |
| MARFaceBeauty | iOS OC | ❌ 仅磨皮 | MIT(已归档) | 最简磨皮 shader |
| PixelFreeEffects | 全平台 | ✅ 功能最全 | **闭源商业SDK,要授权费** | 不自研走付费才用 |
| —— | | | | |
**结论**：没有一个 macOS 原生、能直接合并。可复用的是底层 GLSL shader 算法（要重写成 Metal）。瘦脸大眼依赖人脸关键点 → 我们用 **Apple Vision 框架**（免费、系统自带、76 点）替代 Face++/MediaPipe。

## 五、技术架构（全 Apple 原生 + 免费）
| 层 | 用什么 |
|---|---|
| 地基 | Snapzy |
| 摄像头采集 | AVFoundation（新建） |
| 人脸关键点 | Apple Vision 框架（💡比 MediaPipe 更优，原生免费） |
| 实时美颜 | Metal 着色器（磨皮/美白/瘦脸/大眼/滤镜） |
| 异形露脸框 | Metal 遮罩 + 自定义 Path |
| 隐形提词器 | ScreenCaptureKit 窗口排除 |
| 高清录制 | SCStreamConfiguration Retina 原生分辨率 + 高码率 + H.265/ProRes |
| 救摄像头糊 | iPhone 连续互通相机 + 画质增强 |

## 六、技术验证结果（已跑,带铁证）
- ✅ **提词器命根子**：基座已用 `SCContentFilter(display:excludingApplications:exceptingWindows:)`、`excludingWindows`（ScreenCaptureManager.swift:2047-2049、ScreenRecordingManager.swift:1826）→ "录不进去"基座原生支持，低风险。
- ✅ **摄像头需新建**：基座仅 MicrophoneAudioCapturer（麦克风），无视频摄像头。
- ✅ 录制有 scaleFactor（Retina 相关），高清配置待深入 ScreenRecordingManager。
- ❌ 环境阻塞：未装完整 Xcode（仅命令行工具），磁盘约 45G 偏紧。

## 七、两轮痛点调研 → 差异化金矿
**第一轮**（Loom/Screen Studio/CleanShot 大痛点）：录制崩溃、Screen Studio 没 Windows、录屏软件普遍没美颜、露脸框只有圆/方（心形星形仅 OBS 插件有）、不能自动美化。
**第二轮**（中国创作者特有，欧美工具盲区）：
- ⭐⭐⭐ 竖屏录制几乎没人做（Descript 43 票催更未实现）
- ⭐⭐⭐ 平台安全区遮挡（点赞栏/文案栏盖住内容，转化降~30%）→ 实时安全线参考（几乎无人做）
- ⭐⭐⭐ Mac 内录系统声音要装虚拟声卡（繁琐）
- ⭐⭐⭐ Screen Studio 中文字幕不可用（不断句）
- ⭐⭐⭐ 微信分享链接打不开 / 海外信用卡支付墙 / 国内访问慢
- ⭐⭐⭐ Loom 涨价暴雷 + Screen Studio 取消买断 + 国产"付费才能导出"钓鱼 → 买断制、无水印、不限时、零广告
**用户亲自点破的体感痛点（我调研漏的，最值钱）**：
- 摄像头在屏幕正上方，人看屏幕时脸/眼神偏 → **眼神矫正 Eye Contact + 自动构图**
- Mac 录屏糊 → **高清录制 + iPhone 当摄像头 + 画质增强**
