LumiCue is an invisible macOS teleprompter: a floating glass window for reading scripts while recording, livestreaming, joining calls, or presenting.

LumiCue 是一款 macOS 隐形悬浮提词器。你可以一边录视频、直播、开会或演示，一边看稿；观众、录屏和截图都不会看到提词窗口。

<img src="./poster.png" width="720" alt="LumiCue preview" />

## 中文

### 产品功能

- **隐形悬浮窗口**：提词器只对你可见，不会出现在录屏、直播画面或截图里。
- **全屏演示置顶**：PowerPoint 或演示文稿全屏播放时，提词窗口仍可浮在幻灯片上方。
- **念稿态 / 胶囊态**：大窗口适合完整念稿；窄条胶囊态适合放在镜头附近，减少视线偏移。
- **拖拽缩放**：拖动右下角即可调整窗口大小，文字会随窗口自动适配。
- **进度条跳转**：念稿态和胶囊态都可以拖动进度条，快速跳到稿子任意位置。
- **救场控制**：暂停后可重念当前句、后退多句，或继续播放。
- **粘贴即用**：直接粘贴稿子，也可以导入 `.txt` 文本文件。
- **样式调整**：支持字体、字号和文字颜色切换。
- **语音跟随**：内置离线语音识别，朗读时自动跟随稿件进度。
- **中英界面切换**：控制条可在中文和英文界面之间切换。

### 快捷键

| 操作 | 按键 |
| --- | --- |
| 播放 / 暂停 | `空格` |
| 后退一句 | `↑` |
| 前进一句 | `↓` |
| 粘贴稿子 | `⌘V` |

## English

### Product Features

- **Invisible floating window**: the teleprompter is visible to you, but hidden from screen recordings, livestreams, and screenshots.
- **Full-screen presentation support**: the cue window can stay above PowerPoint or presentation slides.
- **Full mode / Pill mode**: use the full window for comfortable reading, or collapse it into a slim strip near the camera.
- **Drag to resize**: adjust the window from the bottom-right corner, with text adapting to the new size.
- **Progress scrubbing**: drag the progress bar in either mode to jump through the script.
- **Rescue controls**: pause, restart the current line, step back multiple lines, or resume.
- **Paste and go**: paste a script directly or import a `.txt` file.
- **Style controls**: switch fonts, font size, and text colors.
- **Voice following**: built-in offline speech recognition can follow along as you read.
- **Chinese / English UI**: switch the control labels between Chinese and English.

### Shortcuts

| Action | Key |
| --- | --- |
| Play / Pause | `Space` |
| Step back one line | `↑` |
| Step forward one line | `↓` |
| Paste script | `⌘V` |
